Description:
Abstract:
This data set is a polygon coverage of the cumulative thickness of Central-Zone 5, in feet, over extent of the Central-Zone 5 coal and its horizon (stratigraphic equivalent) in Central Texas only. The coverage was produced by gridding more than 350 public and confidential point- source data from the National Coal Resources Data System (NCRDS) of the USGS, using EarthVision (Version 4.0.2) software, subsequently contoured into 6-inch intervals, and imported to ARC/INFO. The 'thickness' item for each polygon reflects a value in feet halfway between the bounding contour lines. This value represents results of gridding which averages point values within cells, therefore, public point data may not always match values represented in the isopach data layer.
Purpose:
This data set was used in the National Coal Resource Assessment Program in the regional evaluation of resources for the coal Central-Zone 5 in the Gulf Coast region. The data set is useful for regional studies- not applicable to local scales.

Data_Quality_Information:
Attribute_Accuracy:
Attribute_Accuracy_Report:
The range of thickness values in the point-source information was 0 to 22.00 feet. The resultant range of the gridded surface was 0-20 feet; extrapolation was used over the x,y extent. The average absolute Z-error of the grid was 1.20 percent of the range.
Logical_Consistency_Report: Topologically clean
Completeness_Report:
Input points were irregularly distributed over the spatial extent of the study, therefore areas of sparse data will not reflect ground truth. Central-Zone 5 cumulative coal thicknesses were selected by name from the stratigraphic data base stored in StratiFact, a relational data base manager in use by the USGS National Coal Assessment. The original data submitted to NCRDS may not have had the Zone identified, a task performed within StratiFact by examining data in cross sections. Data used were selected by geologist from the database.
